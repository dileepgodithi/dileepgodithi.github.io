Academic Projects

Decision Tree Induction

Implemented the ID3 algorithm to induce decision tree using two heuristics (Information gain and variance) on datasets generated synthetically by sampling Boolean CNF. Achieved 98.5% accuracy on 1800 clauses.

Text Classification

Developed an email classification filter using variants of Naïve Bayes and Logistic Regression algorithms. Achieved a maximum accuracy of 99% and a precision of 100% on the Multinomial Naïve Bayes algorithm.

Collaborative Filtering

Developed user-based collaborative filtering (on Netflix dataset) used by recommendation systems. Significantly reduced computation time by using NumPy and data imputation techniques.

Image Compression Using k-means Clustering

Implemented the k-means clustering algorithm to compress images and analyzed the trade-offs between image quality and degree of compression.

Library Management System (LMS)

Designed and developed a native application that interfaces with a backend SQL database implementing LMS. Various modules involve book search, check out/check-in, borrower management and fines management.

Probabilistic Graphical Models - Bayesian Networks

Constructed tree Bayesian Networks using the Chow-Liu algorithm to learn the structure and parameters of the network. Extended models further by mixture of trees using expectation maximization and random forests.
